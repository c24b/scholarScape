Welcome !

You want to crawl Google Scholar the easy way ?

scholarScape allows you to automatically gather data from Google Scholar and then visualize the cited-by relationships between articles and publications.

Get Started !

goal

scholarScape's goal is to help people to understand how academic publications are linked to each other. It is particularly useful for researchers in sociology but it also can benefits PhDs who work a lot with academic articles.

technologies

scholarScape is built upon robust technologies like Scrapy, Twisted and Python.
